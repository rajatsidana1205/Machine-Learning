# Machine Learning & Deep Learning Projects

This repository contains practical **Machine Learning and Deep Learning projects implemented using Python** to solve real-world predictive analytics problems.

The projects demonstrate the end-to-end machine learning workflow including **data preprocessing, feature engineering, model training, evaluation, and optimisation**.

These projects were developed as part of my learning journey in **Artificial Intelligence and Data Analytics** during my **Master of Business Analytics (AI Specialisation) at Deakin University, Australia**.

---

## 🔹 Skills Demonstrated

- Python (Pandas, NumPy)
- Machine Learning (Scikit-learn)
- Deep Learning (TensorFlow / Keras)
- Data preprocessing & feature engineering
- Model training and evaluation
- Time-series prediction
- Natural Language Processing (NLP)
- Model optimisation techniques (Early Stopping)

## 📊 Project 1 – Airport Reviews Sentiment Analysis

**Objective:**  
Analyse airport passenger reviews to determine customer sentiment and identify patterns in passenger feedback.

**Key Questions Answered:**

- How do passengers feel about airport services?
- What are the common positive and negative experiences?
- Can sentiment classification help understand customer satisfaction?

**Techniques Used:**

- Natural Language Processing
- Text preprocessing
- Sentiment classification
- Machine learning models

📄 File: `Airport_Reviews.ipynb`
